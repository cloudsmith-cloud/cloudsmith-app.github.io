Platform Capabilities

Everything your private cloud needs

CloudSmith covers the full infrastructure lifecycle — from day-zero cluster deployment to day-two operations, firmware management, and AI-assisted diagnostics. One platform, every stage.

Core Capabilities

Six disciplines, one platform

Each capability ships as a module with a clean interface slot — compose only what you need, extend with your own modules, or contribute upstream.

⚡

Deploy

Bootstrap Hyper-V failover clusters and Azure Local instances from a browser-based wizard. Define cluster topology, network configuration, and storage layout — CloudSmith handles the rest via a structured installation pipeline with full rollback support.

🖥
Cluster deployment wizard — node discovery, network config, validation
Hyper-V Azure Local WSFC Wizard
📊

Monitor

Real-time health dashboards aggregated from all nodes, storage paths, and workloads. Built-in alerting with configurable thresholds, alert routing, and integration with existing monitoring stacks via OpenTelemetry export.

🖥
Cluster health overview — node status, storage, workload heat map
Health Alerts OpenTelemetry Dashboards
🔄

Update

Orchestrated OS and firmware update pipelines with live-migration awareness. CloudSmith drains workloads before patching each node, validates patch success, and surfaces a full audit trail of every update event across the fleet.

🖥
Update pipeline — node queue, drain status, firmware versions
OS Patches Firmware Drivers Live Migration
🧠

Troubleshoot

AI-assisted diagnostics surface probable root causes from event logs, performance counters, and cluster validation output. Natural-language queries let operators interrogate infrastructure state without deep PowerShell expertise.

🖥
AI diagnostics panel — event correlation, suggested remediation steps
AI Diagnostics Event Logs Root Cause
🔧

Hardware

Vendor-agnostic BMC management for Dell iDRAC, HPE iLO, and Lenovo XCC. Inventory hardware health, power status, and firmware versions across mixed fleets — without logging into each vendor console separately.

🖥
Hardware inventory — BMC health, power, fan/temp across vendors
Dell iDRAC HPE iLO Lenovo XCC BMC
📋

Planning

Capacity planning tools project resource consumption over time using historical usage trends. Model cluster expansions, estimate hardware requirements before procurement, and validate that new workloads fit before deployment.

🖥
Capacity planner — trend lines, growth projections, headroom analysis
Capacity Growth Trends Procurement
